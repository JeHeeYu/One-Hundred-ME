# :100: 백미

## :star: 프로젝트 개요
<b>100일 챌린지를 도와주는 웹앱 어플리케이션</b>

## :calendar: 개발 기간
<b>2023.09.18 ~ 2023.09.24</b>

## :exclamation: 주요 기능
- 기간 별 챌린지 선택 가능
- 선택 및 실패에 대한 스탬프로 사용자에게 알림
- 일일 챌린지 성공 및 실패 등록
- 사진 저장 기능
